# 💫 About Me

**Backend Developer | Java Specialist | BSc Student in Systems Analysis**

Sou um desenvolvedor focado na construção de sistemas de alta disponibilidade e escalabilidade dentro do ecossistema **Java**. Atualmente, curso o 5º período de **Análise e Desenvolvimento de Sistemas na FASEC**, onde sólido conhecimentos em arquitetura de software, padrões de projeto e Clean Code.

### 🚀 Foco Técnico
Minha atuação principal concentra-se no desenvolvimento de **APIs RESTful com Spring Boot**, priorizando a eficiência técnica e a manutenibilidade do código. Possuo um perfil híbrido com forte viés em infraestrutura: como **Power User de Fedora Linux**, utilizo o terminal como ferramenta primária para otimização de fluxos de trabalho, automação e gestão de ambientes.

### 🛠️ Core Stack & Expertise
* **Backend:** Java 17+, Spring Boot (Data, Security, Cloud, MVC), Hibernate/JPA.
* **Web Services:** Arquitetura de microserviços e documentação técnica de APIs.
* **DevOps & Infra:** Administração de sistemas Unix-like (Fedora), Docker, CI/CD e automação.
* **Hardware Enthusiast:** Integração de sistemas e baixo nível utilizando Arduino.

### 🎯 Professional Goals
Busco integrar times de alta performance onde eu possa aplicar soluções de engenharia de software para resolver problemas complexos, contribuindo para o sucesso do produto enquanto evoluo continuamente em arquiteturas distribuídas e computação em nuvem.
